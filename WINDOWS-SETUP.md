# KNX Group Address Studio - Getting Started for Windows\n\n## Quick Start\n\n### Option 1: Download Installer (Easiest)\n1. Go to [Releases](https://github.com/matz-star/KNX-GroupAddress-Studio/releases)\n2. Download `KNX-GroupAddress-Studio-1.0.0.exe`\n3. Run the installer\n4. Launch from Start Menu\n\n### Option 2: Portable Version (No Installation)\n1. Download `KNX-GroupAddress-Studio-1.0.0-portable.exe`\n2. Run directly - no installation needed\n3. Data saves automatically\n\n### Option 3: Development Setup\n\n**Prerequisites**:\n- [Node.js 14+](https://nodejs.org/)\n- [Git](https://git-scm.com/)\n- A code editor (VS Code recommended)\n\n**Steps**:\n```bash\n# Clone repository\ngit clone https://github.com/matz-star/KNX-GroupAddress-Studio.git\ncd KNX-GroupAddress-Studio\n\n# Install dependencies\nnpm install\n\n# Start development\nnpm run dev\n```\n\n## First Use\n\n1. **Launch the app** - Green/Blue interface appears\n2. **Name your project** - Click \"Rename\" in header\n3. **Add addresses** - Click \"Add New\" button\n4. **Fill the form** - Address, Name, Description, DPT, Comment\n5. **Save** - Click \"Add\"\n6. **Export** - Click \"Export CSV\" when ready\n\n## Import/Export\n\n### Export to CSV (For ETS6)\n```\n1. Click \"Export CSV\" button\n2. File saved as: ProjectName_2024-01-15.csv\n3. Open ETS6 → Import this CSV\n```\n\n### Import from CSV\n```\n1. Click \"Import CSV\" button\n2. Select your CSV file\n3. Duplicates automatically skipped\n4. New addresses added\n```\n\n## CSV Format\n\n```csv\nAddress,Name,Description,DataPointType,Comment\n1/0/1,Light_Kitchen,Kitchen main light,1.001,Switch\n1/0/2,Light_Bath,Bathroom light,1.001,Switch\n```\n\n**Important**:\n- Save as UTF-8 encoding\n- Headers must match exactly\n- Address format: `X/Y/Z`\n- DPT format: `X.XXX`\n\n## Building for Distribution\n\n### Create Installer (.exe)\n```bash\nnpm run dist\n```\nOutput: `dist/KNX-GroupAddress-Studio-1.0.0.exe`\n\n### Create Portable (.exe)\n```bash\nnpm run dist:portable\n```\nOutput: `dist/KNX-GroupAddress-Studio-1.0.0-portable.exe`\n\n## Keyboard Shortcuts\n\n- `Ctrl+Q` - Quit application\n- `Ctrl+E` - Export (after setup)\n- `Ctrl+I` - Import (after setup)\n\n## Windows Registry\n\nApplication settings stored in:\n```\nHKLM\\Software\\Policies\\KNX-GroupAddress-Studio\n```\n\nUser data stored in:\n```\n%APPDATA%\\KNX-GroupAddress-Studio\n```\n\n## Uninstall\n\n**Via Control Panel**:\n1. Settings → Apps → Apps & features\n2. Search for \"KNX Group Address Studio\"\n3. Click → Uninstall\n\n**Portable Version**:\n- Simply delete the .exe file\n\n## Troubleshooting\n\n### \"Cannot find module\" error\n```bash\n# Clear cache and reinstall\nrm -r node_modules package-lock.json\nnpm install\n```\n\n### Application crashes on startup\n- Reinstall from fresh clone\n- Check Windows 10/11 is up to date\n- Ensure .NET Framework 4.5+ installed\n\n### Data not saving\n- Check AppData folder has write permissions\n- Try portable version\n- Run as Administrator\n\n### Port 3000 in use (Dev only)\n```bash\nnetstat -ano | findstr :3000\ntaskkill /PID <PID> /F\nnpm run dev\n```\n\n## System Requirements\n\n- **OS**: Windows 10 or Windows 11 (x64)\n- **RAM**: 1GB minimum\n- **Disk**: 100MB free space\n- **.NET**: Framework 4.5+ (for installer)\n- **Processor**: x64 compatible\n\n## Next Steps\n\n- 📖 Read full [README.md](README.md)\n- 🐛 Report issues on [GitHub](https://github.com/matz-star/KNX-GroupAddress-Studio/issues)\n- ⭐ Star the project if you find it useful\n- 🔧 Contribute improvements\n\n## Support\n\nFor help:\n1.
